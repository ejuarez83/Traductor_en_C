Tareas para que funcione el traductor

1. Copiar la carpeta voices a c:\lib\voices
-En caso de omitir este paso, el programa intentara generar un archivo cada vez que se traduzca una palabra

2. Copiar a la carpeta c:\lib los archivos:
-1000.wav
-2000.wav
-2001.wav
-3000.wav
estos son audios de menu

3. Si se desea recompilar se debe tener instalado Boost como libreria
4. Para correrlo se debe tener instalado el interprete de python 3.10 en adelante
5. Se debe agregar en el linker del proyecto -lwinmm
6. Se debe usar toda la carpeta del proyecto.
